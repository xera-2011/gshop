# 硅谷外卖

## 项目工具
> 打包工具webpack4.0,Vue框架
> axios、vue-router、vuex
> mint-ui组件库、swiper轮播数据、better-scroll移动端滚动插件、date-fns日期工具、fastclick清除点击延迟、mockjs拦截 Ajax 请求
